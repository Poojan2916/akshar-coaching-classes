📖 What is a Linear Equation in One Variable?

A linear equation in one variable is an equation of the form ax + b = 0, where:

  • a and b are real numbers, a ≠ 0
  • x is the unknown variable
  • It always has exactly one unique solution
  • The solution is the value of x that makes LHS = RHS
✅ Key Fact: "Linear" means the highest power of the variable is 1. So x² makes it quadratic, not linear.
🔄 The Golden Rules of Solving

Whatever you do to one side, do the same to the other side.

OperationRuleExample
Add/SubtractAdd/subtract same number from both sidesx − 3 = 7 → x = 7 + 3 = 10
Multiply/DivideMultiply/divide both sides by same non-zero numberx/4 = 5 → x = 20
TransposeMove term to other side, flip sign (+ → −, × → ÷)2x + 5 = 11 → 2x = 11 − 5 = 6
Cross-multiplyFor a/b = c/d → a × d = b × c(x+1)/(2x+3) = 3/8 → 8(x+1) = 3(2x+3)
📋 Step-by-Step Method to Solve Any Equation
1

Clear fractions: Find the LCM of all denominators and multiply both sides by it.

2

Expand brackets: Use distributive property: a(b + c) = ab + ac

3

Collect variables: Move all x terms to the LEFT side.

4

Collect constants: Move all numbers to the RIGHT side (flip signs when crossing = ).

5

Divide: Divide both sides by the coefficient of x to get x = answer.

6

Verify: Substitute back into the ORIGINAL equation — LHS must equal RHS.

🔍 Worked Examples (Fully Solved)
Example 1: Solve 2x/3 + 1 = 7x/15 + 3
LCM of 3 and 15 = 15. Multiply throughout by 15:
→ 10x + 15 = 7x + 45
→ 10x − 7x = 45 − 15 → 3x = 30 → x = 10
Example 2: A father is 30 years older than his daughter. After 5 years he will be 3× her age.
Let daughter's present age = x. Father's age = x + 30.
After 5 years: x + 35 = 3(x + 5)
→ x + 35 = 3x + 15 → 20 = 2x → x = 10 (Daughter = 10, Father = 40)
Example 3: 2-digit number. Sum of digits = 9. Reversed number exceeds original by 45.
Let tens digit = x. Units = 9 − x. Original = 9x + 9. Reversed = 90 − 9x.
→ (90 − 9x) − (9x + 9) = 45 → 81 − 18x = 45 → 18x = 36 → x = 2
→ Number = 27
Example 4: Ages in ratio 3:4. Six years ago ratio was 7:10. Find present ages.
Let present ages = 3x and 4x.
Six years ago: (3x − 6)/(4x − 6) = 7/10
→ 10(3x − 6) = 7(4x − 6) → 30x − 60 = 28x − 42 → 2x = 18 → x = 9
→ Ages = 27 and 36
🔢 Essential Formula Sheet
Standard Form: ax + b = c → x = (c − b) / a
Cross-multiplication: a/b = c/d → ad = bc
Clearing Fractions: Multiply both sides by LCM of all denominators
Distributive Law: a(b ± c) = ab ± ac
Transposition: ax + b = c → ax = c − b → x = (c−b)/a
📊 2-Digit Number Formulae
If tens digit = x, units digit = y:
Original number = 10x + y
Reversed number = 10y + x
Sum of digits = x + y
Difference (original − reversed) = 9(x − y)
Sum (original + reversed) = 11(x + y)
💡 If digits sum to S and tens > units by D: tens = (S+D)/2, units = (S−D)/2
⏱️ Age Problem Formulae
Present age = x
Age n years LATER = x + n
Age n years BEFORE = x − n

If A is k times older than B now: A = kB
If after n years A will be k times B: (A+n) = k(B+n)
Ratio problem: If ages are in ratio p:q → let ages = px and qx
Apply past/future condition and solve for x.
🚤 Distance–Speed–Time
Distance = Speed × Time
Speed = Distance / Time
Time = Distance / Speed

Downstream speed = boat speed + stream speed
Upstream speed = boat speed − stream speed

If same distance: Time₁ × Speed₁ = Time₂ × Speed₂

💡 Exam Tricks & Smart Strategies
🎯 Trick 1 — LCM shortcut: When you see fractions like x/2 + x/3, find LCM (=6) and multiply both sides. All denominators vanish instantly.
🎯 Trick 2 — Cross-multiply when one fraction = one fraction: If the equation is a/b = c/d, immediately cross multiply. Don't find LCM.
🎯 Trick 3 — Digit problem shortcut: Original + Reversed = 11(sum of digits). Original − Reversed = 9(difference of digits). Use these to set equations fast.
🎯 Trick 4 — Always verify! In exams, the last step "Check: LHS = RHS" gets you marks even if you made a small calculation error.
🎯 Trick 5 — Word problem shortcut: Read the LAST line of the question first — it tells you what the equation should equal.
⚠️ Common Mistakes to AVOID
❌ Mistake 1: Forgetting to change sign when transposing. If +3 crosses = it becomes −3. Always flip the sign!
❌ Mistake 2: Not multiplying ALL terms by the LCM. Every single term on both sides must be multiplied.
❌ Mistake 3: In age problems — forgetting to add the same years to BOTH people. If "after 5 years", both ages increase by 5.
❌ Mistake 4: Confused about "exceeds by" vs "is half of." Read carefully: "A exceeds B by 5" means A = B + 5, NOT B = A + 5.
❌ Mistake 5: In digit problems — the "original number" is 10×(tens digit) + units digit. Don't just add the digits.
❌ Mistake 6: Skipping verification. Even a simple "LHS = 7 = RHS ✓" at the end earns marks.

🧩 Word Problem Practice Bank
Age Problem

1. A father is 30 years older than his daughter. After 5 years, the father will be 3 times her age. Find their present ages.

Let daughter = x. Father = x + 30.
After 5 years: x + 35 = 3(x + 5) → x + 35 = 3x + 15 → 2x = 20 → x = 10
✅ Daughter = 10 years, Father = 40 years.
Age Ratio

2. Sham and Sadhu's ages are in ratio 3:4. Six years ago, ratio was 7:10. Find present ages.

Let ages = 3x and 4x.
(3x−6)/(4x−6) = 7/10 → 30x−60 = 28x−42 → 2x = 18 → x = 9
✅ Sham = 27, Sadhu = 36
Digit Problem

3. Sum of digits of a 2-digit number = 11. The number exceeds its reverse by 45. Find the number.

Tens = x, Units = 11−x. Original = 9x+11, Reversed = 110−9x.
(9x+11)−(110−9x) = 45 → 18x−99 = 45 → 18x = 144 → x = 8
✅ Number = 83
Number Problem

4. One number is twice another. If 30 is subtracted from both, one becomes 4 times the other. Find the numbers.

Let numbers = 2x and x. (2x−30) = 4(x−30) → 2x−30 = 4x−120 → 90 = 2x → x = 45
✅ Numbers = 45 and 90
Speed Problem

5. A boat covers the distance between two points downstream in 4 hours and upstream in 5 hours. Stream speed = 2 km/hr. Find boat speed.

Let boat speed = x km/h. Downstream speed = x+2, Upstream = x−2.
Distance same: 4(x+2) = 5(x−2) → 4x+8 = 5x−10 → x = 18
✅ Boat speed = 18 km/h
Fraction Problem

6. Denominator of a rational number is greater than numerator by 8. If numerator increases by 17 and denominator decreases by 1, number = 3/2. Find original number.

Let numerator = x. Denominator = x+8.
(x+17)/(x+8−1) = 3/2 → 2(x+17) = 3(x+7) → 2x+34 = 3x+21 → x = 13
✅ Fraction = 13/21
Perimeter

7. Perimeter of a triangle is 62/3 cm. Two sides are 2½ cm and 3⅓ cm. Find the third side.

Third side = 62/3 − 5/2 − 10/3
= 62/3 − 10/3 − 5/2 = 52/3 − 5/2 = 104/6 − 15/6 = 89/6 = 14⅚ cm
Toffee Distribution

8. Ankur gives 1/3 to Ravi, 1/4 to Amit, 1/6 to Sunil, and keeps 15. How many toffees did he start with?

Let total = x.
Given away: x/3 + x/4 + x/6 = (4x+3x+2x)/12 = 9x/12 = 3x/4
Remaining: x − 3x/4 = x/4 = 15 → x = 60 toffees
